🏥 직원 건강검진, 이렇게 관리하면 편합니다 — Medical Map 소개

회사에서 건강검진 담당자라면 공감하실 겁니다.
수십 개 검진기관 중 어디가 우리 사무실과 가장 가까운지, 어느 기관이 어떤 지역을 커버하는지… 매번 지도 앱을 열고 하나하나 확인하는 게 정말 번거롭죠.

그래서 만들었습니다. 건강검진 기관과 고객사(기업)를 한 지도 위에서 한눈에 관리하는 웹 서비스 — Medical Map 입니다.

📍 이런 분들에게 필요합니다

  • 기업 HR / 보건관리팀에서 직원 건강검진을 담당하는 분
  • 다수의 계약 검진기관을 지역별로 관리해야 하는 분
  • "우리 사무실 근처 검진기관이 어디야?"라는 질문을 자주 받는 분
  • 검진기관 정보를 엑셀 대신 지도로 시각화하고 싶은 분

🗺️ 주요 기능 소개

① 지도 기반 검진기관 + 고객사 동시 조회

카카오맵 위에 검진기관(파란 원형 마커)과 고객사(노란 핀 마커)를 함께 표시합니다. 각 레이어를 켜고 끌 수 있어서, 원하는 정보만 골라 볼 수 있습니다. 기관 마커를 클릭하면 이름·주소·그룹 정보가 팝업으로 표시됩니다.

② 지역 / 그룹 필터링 & 실시간 검색

시·도 → 시·군·구 2단계 드롭다운으로 원하는 지역의 기관만 필터링합니다. 그룹별 색상 필터로 계약 종류나 등급을 구분해 표시할 수도 있습니다. 검색창에 기관명을 입력하면 지도가 해당 위치로 이동합니다.

③ 반경 검색 — 근처 기관 TOP 5 & 거리 계산

특정 고객사(사무실)를 선택하고 반경 슬라이더(1~50km)를 조절하면, 해당 반경 내의 검진기관이 거리순으로 정렬되어 표시됩니다. 반대로 검진기관 기준으로 주변 고객사를 찾는 것도 가능합니다. 지도에는 점선 원으로 검색 범위가 시각화되어 직관적입니다.

④ 목록 관리 (추가 / 수정 / 삭제)

목록 페이지에서 검진기관과 고객사 정보를 직접 추가하거나 수정할 수 있습니다. 주소를 입력하면 카카오 API로 좌표를 자동으로 가져오기 때문에, 위도·경도를 직접 입력할 필요가 없습니다. 가나다순·지역·주소 기준 정렬과 30개씩 페이지네이션도 지원합니다.

⚙️ 기술 스택

항목 내용
프론트엔드 HTML5, CSS3, Vanilla JavaScript (ES6+)
지도 API 카카오맵 JavaScript SDK (마커, 클러스터, 지오코딩)
데이터 저장 LocalStorage (변경이력 관리) + JSON 파일
거리 계산 Haversine 공식 (위도·경도 기반 대원거리)

📱 모바일에서도 편하게

반응형 디자인으로 제작되어 스마트폰(768px 이하)에서도 탭 기반 UI로 전환됩니다. 사무실 밖에서도 어디서나 검진기관 정보를 확인할 수 있습니다.

🚀 이렇게 사용해보세요

  1. 지도 페이지(index.html)를 열어 전체 기관 분포를 한눈에 파악합니다.
  2. 왼쪽 패널에서 시/도 → 시/군/구를 선택해 원하는 지역으로 좁혀봅니다.
  3. 반경 검색(nearby.html)으로 특정 사무실 기준 가장 가까운 검진기관을 찾습니다.
  4. 목록 페이지(list.html)에서 새로운 기관을 추가하거나 기존 정보를 수정합니다.
  5. 그룹 기능으로 계약 등급·지역별 색상 분류를 적용해 관리 효율을 높입니다.
